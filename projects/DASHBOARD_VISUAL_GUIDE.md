### 2. Card Hover Effect
```
Before Hover:
  - Y Position: 0px
  - Shadow: Default
  - Border Color: Lighter

On Hover:
  - Y Position: -8px (lifts up)
  - Shadow: Larger, darker
  - Color: Slightly brighter
  - Duration: 0.3s smooth
```

### 3. Button Hover Effect
```
Before Hover:
  - Background: White
  - Border: Gray
  - Shadow: Subtle

On Hover:
  - Background: Gradient color (blue/green/orange/purple)
  - Text Color: White
  - Shadow: Colored glow
  - Duration: 0.3s smooth
```

### 4. Status Indicator Pulse
```
Stage 1: Green dot appears

  - Opacity: 100% â†’ 0%
```
